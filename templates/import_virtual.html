Import marketing registrations into virtual_main_data_center_registrations (Vision export, same column layout as in-person Main Data Center). Challenge credits still use the JSON API below.

Challenge submissions (XLSX)

How this import works

  • Destination table: virtual_challenge_submission_rows (one row per team per challenge, linked to challenges and Virtual MDC).
  • Upsert: same challenge + same team name → existing row is updated; new team names → inserted. Nothing is deleted.
  • Sheets: every tab must look like Submission … (optional 1. / 2) prefix is ignored). The suffix must match the challenge title or the optional sheet suffix in Manage challenges.
  • Gate: each Leader email must already exist in virtual_main_data_center_registrations for event {{ virtual_event_id }}. If any email is missing, no rows are written (fix MDC, then retry).

Only .xlsx is supported. After you run an import, the panel below shows whether the database was updated, which tabs were matched, and row counts.
